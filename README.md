# hunnu-venue-booking-script
基于湖师大往期预约系统的校园场地预订脚本的整理回顾。本仓库仅用于学习目的，为历史系统的复盘归档。所涉及流程与代码已全部失效，不具备任何现实可用性。
项目不再维护，也不接受功能性相关问题。
## 说明
本项目是本人在对**湖师大旧版羽毛球场地预约系统**进行的流程复盘与代码整理。
**注意：**
- 所涉及系统已升级，相关流程已失效
- 本仓库代码不可用于任何真实预约系统
- 项目不再维护，仅用于学习参考
---
## 内容
本项目记录并整理了旧系统下的：
- 预约流程的 HTTP 请求顺序  
- 前端操作与后端接口的对应关系  
- 简单自动化脚本的组织方式  
重点在于流程分析，而非功能可用性。
